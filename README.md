A mod for `Monster Hunter Stories 3` that keep all monsties in normal size. / 用于`怪物猎人物语3`的锁定随行兽为正常体型MOD。

> [!NOTE]
> These two versions are **mutually exclusive alternatives** and they **do not depend on each other**. Please choose ONLY ONE version to install. / 
> 这两个版本的 MOD 是**二选一**的关系，且**互不依赖**，请仅挑选其中**一个**版本进行安装即可。

---

## Version A: REFramework Lua Script Version

### Requirements:

- [REFramework](https://github.com/praydog/REFramework-nightly/releases) (≥ Nightly 01298)

### Mod Download:

[https://github.com/dzxrly/MHST3-NormalOtomonScale/releases/](https://github.com/dzxrly/MHST3-NormalOtomonScale/releases/) (e.g., `NormalOtomonScale_v...zip`)

### How To Install:

1. Install [REFramework](https://github.com/praydog/REFramework-nightly/releases) (≥ Nightly 01298);
2. Download the Lua script version of the mod;
3. Unzip the mod (**ALL FILES**) to the Monster Hunter Stories 3 **ROOT** directory.
4. When you install correctly, there will be a directory and a file in your `./reframework`, and they are:

    - `./reframework/autorun/normal_otomon_scale.lua`
    - `./reframework/autorun/NormalOtomonScale`

---

## Version B: Native `user.3` Files Version

### Requirements:

- [REFramework](https://github.com/praydog/REFramework-nightly/releases) (≥ Nightly 01298)
- Ensure that the **LooseFileLoader** feature is enabled in REFramework to load native files.

### Mod Download:

[https://github.com/dzxrly/MHST3-NormalOtomonScale/releases/tag/user3-latest](https://github.com/dzxrly/MHST3-NormalOtomonScale/releases/tag/user3-latest) (Rolling Release for Native PC)

### How To Install:

1. Install [REFramework](https://github.com/praydog/REFramework-nightly/releases) and enable **LooseFileLoader**;
2. Download the zip file;
3. Install via Fluffy Mod Manager, OR extract the `natives` folder directly into your Monster Hunter Stories 3 **ROOT** directory.

---

### MOD下载:

[https://github.com/dzxrly/MHST3-NormalOtomonScale/releases/tag/user3-latest](https://github.com/dzxrly/MHST3-NormalOtomonScale/releases/tag/user3-latest) （这是专属的滚动更新发布页面）
