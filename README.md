A mod for `Monster Hunter Stories 3` that keep all monsties in normal size. / 用于`怪物猎人物语3`的锁定随行兽为正常体型MOD。

> [!NOTE]
> The mod now comes in two variants. They are **mutually exclusive alternatives**. Please choose ONLY ONE version to install. / 
> 此 MOD 现在提供两个独立的变体版本。它们是**二选一**的关系，请仅挑选其中**一个**压缩包进行安装即可。
>
> 1. **Normal Version (`NormalOtomonScale_v...zip`)**: Keeps all monsties at standard 1.0x size. / 基础版本：将所有随行兽体型锁定为标准的 1.0 倍。
> 2. **With Enemy Size (`NormalOtomonScale_with_enemy_size_v...zip`)**: Calculates and applies original wild enemy scaling. / 应用敌人缩放版本：根据野外同类怪物的体型缩放数据，计算并应用不同的体型比例。

---

## Requirements:

- [REFramework](https://github.com/praydog/REFramework-nightly/releases) (≥ Nightly 01298)
- Ensure that the **LooseFileLoader** feature is enabled in REFramework to load native files.

## Mod Download:

[https://github.com/dzxrly/MHST3-NormalOtomonScale/releases/latest](https://github.com/dzxrly/MHST3-NormalOtomonScale/releases/latest)

## How To Install:

1. Install [REFramework](https://github.com/praydog/REFramework-nightly/releases) and enable **LooseFileLoader**;
2. Download one of the zip files based on your preference (Standard vs With Enemy Size);
3. Install via Fluffy Mod Manager (Recommended), OR extract the `natives` folder directly into your Monster Hunter Stories 3 **ROOT** directory.
